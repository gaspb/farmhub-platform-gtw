A HJL Sensor is only defined by it's acquisition driver and data format.
The driver is the code that reads from the hardware GPIO and formats the data.
Data format should be documented for a better integration with in-app tools.

A HJLSensor can be shared with your team and the community.
New drivers can be loaded at runtime, allowing hardware modification without downtime or redeployment
